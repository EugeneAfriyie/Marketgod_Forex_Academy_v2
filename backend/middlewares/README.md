## 🗄️ Models Folder

**What is this?**
This folder contains the "blueprints" for your database using Mongoose.

**What goes here?**
Files that define what data a specific entity can hold. 
- `User.js` (Defines that a user has a name, email, password, and MarketGod Coins).
- `Course.js` (Defines a course title, price, and video links).
